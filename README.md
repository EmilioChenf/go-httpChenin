API Champions League

API REST construida en Go utilizando la libreria estandar.

Tema:
Ultimos campeones de la UEFA Champions League.

Endpoints

GET /api/items
Obtiene todos los campeones

GET /api/items?id=1
Obtiene campeon por query

GET /api/items/1
Obtiene campeon por path

POST /api/items
Crear campeon

PUT /api/items/{id}
Actualizar campeon

DELETE /api/items/{id}
Eliminar campeon
